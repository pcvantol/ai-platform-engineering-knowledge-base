# Capability Examples

This area points to example capability definitions.

Capability examples are non-active. They do not register, activate or implement runtime capabilities.

## Examples

See [../../capabilities/examples.md](../../capabilities/examples.md).

Example capability families:

- Verification Engineering;
- Software Assurance;
- Architecture Engineering;
- Release Engineering.

## Boundary

Examples do not:

- implement capabilities;
- activate capabilities;
- modify Certified Knowledge;
- modify runtime architecture;
- approve governance;
- register production extensions.
