# Goal Orchestration Examples

This area points to example engineering goals and goal plans.

Goal examples are non-executable. They do not implement `aikb goal`, run agents, modify repositories or create deliverables.

## Examples

See [../../goal-orchestration/examples.md](../../goal-orchestration/examples.md).

Example goals:

- Create a verification platform;
- Bootstrap an embedded AI client;
- Introduce Software Assurance.

## Boundary

Examples do not:

- execute runtime tasks;
- implement AI agents;
- modify Certified Knowledge;
- approve engineering work;
- invent engineering evidence.
