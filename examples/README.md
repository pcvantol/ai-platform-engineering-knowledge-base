# Examples

This area will contain future examples for consuming the Knowledge Base through the CLI, generators, templates, and derived outputs.

Examples are non-canonical. They may demonstrate usage, but they must not become source knowledge.

This area does not contain generated example repositories yet. Runnable examples include:

```text
bin/aikb init verification-platform --name example-verification-platform --output /tmp/example-verification-platform
```

```text
bin/aikb ask "How should I architect a verification runtime?"
```

## Example Principles

Examples may show:

- conceptual CLI usage;
- generated repository layouts;
- onboarding packet shapes;
- publication generation flows;
- question answering flows;
- health report examples;
- traceability reporting examples.

Examples must:

- reference source Certified Knowledge when demonstrating knowledge use;
- remain clearly derived;
- avoid project-specific assumptions unless the example declares them;
- avoid becoming implementation requirements.
