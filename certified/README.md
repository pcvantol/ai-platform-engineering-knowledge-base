# Certified Knowledge

This area contains certified knowledge.

Certified knowledge is the highest authoritative form inside the Knowledge Base. It is reviewed, approved, reusable, and suitable as a source for publications.

Certified knowledge remains canonical even when it is later used to generate or support multiple publications.

The canonical process for promoting knowledge into this area is defined in [../certification/README.md](../certification/README.md).

The operational certification command is `aikb certify`, documented in [../cli/README.md](../cli/README.md). It transforms eligible Generalized Knowledge into Certified Knowledge and records certification history.
