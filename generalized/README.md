# Generalized Knowledge

This area contains generalized knowledge.

Generalized knowledge is project-independent knowledge abstracted from one or more knowledge concepts. It must not depend on DJConnect, Verification Runtime, or any other specific software system.

Generalized knowledge is not authoritative until it becomes certified knowledge.

The operational generalization command is `aikb generalize`, documented in [../cli/README.md](../cli/README.md). It transforms Knowledge Concepts into Generalized Knowledge only.
